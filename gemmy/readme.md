Some pretty cool things simple form does for basic form functionality:


+Hints and better labels are a breeze to add
+Default input fields can be changed easily
+Simply add ranges to dropdowns





**It's also compatable with bootstrap! Hooray!

Let's install it:

In your Gemfile:
gem 'simple_form'
Run the following command to install it:
$bundle install
Run the generator:
$rails generate simple_form:install
