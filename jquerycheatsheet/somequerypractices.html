The actual code:

$(document).ready(
	function(){
		$("#slideUpDemo").click(
			function() {
				$("#slideUpDemo").slideUp();
			}
		);

		$("#slideUpDemo").click(
			function() {
				$("#slideUpDemo").slideDown();
			}
		);

		$("#blur").click(
			function() {
				$("#blur").fadeOut(7000); //and hey look I changed the timing of the fade
			}
		);


				$("#mouseleave").mouseleave(
			function() {
				$("#mouseleave").fadeOut(500); //and hey look there's a mouse event that isn't clicking
			}
		);
	}
)

So here's the code in Sublime Text 3, which is sublime because it color codes certain pieces of the code for you to recognize.
  • Pink just shows the cute dollar signs, the name of the main funtion in jQuery.
  • Yellow is showing us strings.
  • Purple is showing us number-literals and can be decimal, binary, octal, and hexadecimal.
  • Blue is showing reserved words and cannot be used as variables or function names.
